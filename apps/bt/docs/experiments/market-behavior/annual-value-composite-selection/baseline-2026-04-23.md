## Main Conclusion

This should be treated as a new research branch, and the branch is useful.

The strongest practical region is still `standard` stocks with no hard market
cap floor. `ADV60 >= 10mn JPY` keeps a meaningful return profile, but any
`market cap >= 10bn JPY` floor removes a large part of the effect. That means
the signal is not simply "cheap stocks"; it is specifically cheap small stocks,
with low `forward PER` adding a useful value tie-break.

The read is that `EPS > 0` should not be a default alpha filter for the
`55/25/20` style composite. It likely removes some small turnaround candidates
where latest FY EPS is still negative but forward EPS is positive. It can still
be useful as a conservative portfolio variant when avoiding loss-making
companies is more important than maximizing the discovered effect.

## Practical Read

Best next branch:

1. `standard`
2. low `PBR`
3. small market cap
4. low `forward PER`
5. `ADV60 >= 10mn JPY` as the first capacity floor
6. avoid a hard `market cap >= 10bn JPY` floor unless the objective changes
   from alpha discovery to deployable-capacity screening

The score-based version confirms the earlier rule-based result. The main open
work is production realism: execution timing, slippage, position caps, turnover,
and whether a softer capacity penalty beats hard floor filters.

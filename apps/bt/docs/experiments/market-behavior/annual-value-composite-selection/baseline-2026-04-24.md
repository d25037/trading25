## Main Conclusion

The practical `PBR > 0` and `forward PER > 0` rerun keeps the same structure as
the earlier baseline. The effect is still a `standard` small-value effect, not
just a generic cheap-stock effect.

The opportunity set shrinks by about `26%`, and the broad composite loses some
return, but the optimal branch does not change:

1. `standard`
2. small market cap first
3. low `PBR` second
4. low `forward PER` as the additive tie-break
5. do not include an `ADV60` floor in the deployable score because it clearly
   lowers return in the capacity check
6. avoid a hard `market cap >= 10bn JPY` floor unless the objective is
   explicitly capacity, not alpha

## Practical Read

This rerun is more realistic and still leaves a usable signal family.

The next deployable screen should be closer to:

1. `standard`
2. `score = 55% * small cap + 25% * low PBR + 20% * low forward PER`
3. `PBR > 0`
4. `forward PER > 0`

That keeps the basic alpha shape while removing the least interpretable
negative-ratio tail. `ADV60` is intentionally excluded from the score and from
the default eligibility filter because the research table shows a clear return
drag when it is used as a floor.

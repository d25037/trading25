## 解釈

### 1. NT 倍率急低下日の翌朝は broad に強い

- `NT return <= μ-2σ` の翌朝は、全グループで平均 overnight return が大きくプラス。
- 読み: `TOPIX 優位が急に進んだ日` の翌朝は、株式全体でギャップアップが出やすい。

### 2. NT 倍率急上昇日の翌朝は broad に弱い

- `NT return >= μ+2σ` の翌朝は、全グループで平均 overnight return がマイナス。
- 読み: `日経平均優位が急に進んだ日` の翌朝は、翌営業日の寄り付きで反動が出やすい。

### 3. TOPIX100 は tail sensitivity が最も大きい

- `TOPIX100` は `<= μ-2σ` で `0.4130%`、`>= μ+2σ` で `-0.1231%` と、最も振れ幅が大きい。
- 読み: index concentration の強い大型株群は、NT の相対回転に最も素直に反応する。

### 4. PRIME ex TOPIX500 は extremes より平常帯寄与が強い

- `PRIME ex TOPIX500` は extremes でも方向は同じだが、総合期待値の中心は `μ-1σ ~ μ+1σ` にある。
- 読み: sharp な tail bet より、日常的な overnight drift を取りにいくときに相対的に安定している。

### 5. 中央 bucket が総合期待値を支配する

- 中立帯は平均自体は極端 bucket より小さいが、頻度が圧倒的に高いため寄与の総量では最大。
- 読み: tail だけ見て戦略化すると誤解しやすく、頻度加重の視点が必須。

## 注意点

- これは equal-weight のグループ平均であり、コスト控除後の実運用戦略ではない。
- 対象は `event close -> next open` に限定しており、その後の intraday exit は含まない。
- extreme bucket は件数が少ないため、tail 解釈は期間分割で再確認すべき。

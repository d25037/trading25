## TOPIX100 Focused Read

- TOPIX100 は `常に strongest` なわけではなく、`joint regime が噛み合ったときだけ strongest` になります。全体 1 位の `0.1237%` も、central-central の積み上げではなく、少数の強い regime が引っ張っています。
- もっとも実戦的な主力 regime は `NT central x TOPIX +1σ~+2σ` です。日数が `155` 日と十分にあり、平均 `1.3835%`、寄与 `0.0919%` と、TOPIX100 の優位の大半をここで稼いでいます。
- もう一つの勝ち筋は `TOPIX <= -2σ x NT <= μ-1σ` の panic rebound です。件数は `16` 日と少ない一方、平均 `1.7263%` と非常に大きく、`市場は急落しているが relative には TOPIX 側` というときだけ限定的に使う regime です。
- いちばん危険なのは `TOPIX <= -2σ x NT +1σ~+2σ` です。市場全体は弱いのに日経優位が進んでいると、TOPIX100 は `-2.5265%` まで崩れます。これは `broad weakness + mega-cap defensive flow` を真正面から受ける形です。
- `TOPIX >= +2σ` も追いかけてはいけません。`NT central x TOPIX >= +2σ` で `-0.5878%`、`NT >= μ+2σ x TOPIX >= +2σ` でも `-0.3665%` なので、TOPIX100 の本命は `適度に強い market` であって `吹き上がり局面` ではありません。

## 解釈

### 1. baseline は central-central だが、そこで勝つ group は分かれる

- 最頻出は `NT central x TOPIX central` で、`PRIME ex TOPIX500` の寄与が最も大きい。
- 一方 `TOPIX100` はこの cell 自体はそこまで強くなく、別の regime で勝つ。
- 読み: `PRIME ex TOPIX500` は日常 drift 型、`TOPIX100` は regime-sensitive 型。

### 2. `NT central x TOPIX +1σ~+2σ` が最も実戦的な strong regime

- 日数が `155` 日ありつつ、`TOPIX100` の平均 overnight return は `1.3835%`。
- 頻度加重寄与でも `0.0919%` と全 cell 中最大。
- 読み: broad market が「強すぎない強さ」で引け、かつ NT 回転が暴れていない日は、翌朝の continuation を素直に取りやすい。

### 3. `TOPIX <= -2σ` でも NT が弱いなら reversion が勝つ

- `TOPIX <= -2.32%` かつ `NT <= μ-1σ` では、全 group が `1.4% ~ 1.75%` の大幅プラス。
- 読み: 市場全体は急落していても、相対的には TOPIX 側が主導している panic は、翌朝の broad rebound に繋がりやすい。

### 4. `TOPIX <= -2σ` なのに NT が強いと worst regime

- `TOPIX <= -2.32%` かつ `NT +1σ~+2σ` は全 group で `-2.25% ~ -2.53%`。
- 読み: broad market が崩れる一方で日経平均優位が進んでいる日は、mega-cap defensive flow が強く、翌朝の broad long が最も危険。

### 5. `TOPIX >= +2σ` は single-factor と同じく吹き上がり警戒

- `NT central x TOPIX >= +2σ` でも全 group がマイナスで、`TOPIX100` は `-0.5878%`。
- 読み: joint で見ても、強すぎる引けは continuation より mean reversion に寄りやすい。

## 注意点

- これは equal-weight の group 平均であり、コスト控除後の実運用戦略ではない。
- TOPIX close 側は `0%` 基準の `±σ` 分割、NT 側は mean-centered `μ±σ` 分割である。
- 25 cell の一部は日数が少ないため、極端セルの平均値は今後 rolling / out-of-sample で再確認が必要。

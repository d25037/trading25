## Read

- `tradeable definition` に落としても、
  `0-10%` と `10-20%` はどちらも median trade return がプラスでした。
- ただし意味はかなり違います。
  - `0-10%` は path-wise にも `+2.4%` の asymmetry があり、
    `downside より upside がやや優勢` と言いやすいです。
  - `10-20%` は median trade return 自体は `+2.0%` ですが、
    asymmetry は `+0.2%` しかなく、
    `target exit が効くから勝っている` 色が強いです。
- deepest-family 整合で見ると、
  `0-10% entry` は `73.2%` がそのまま `0-10% family` に残り、
  `-7.9%` の悪い `<0% family` は `26.8%` に限られます。
  したがって `0-10%` は ex post family とかなり整合しています。
- 一方 `10-20% entry` は
  `41.6%` しか `10-20% family` に残らず、
  `45.9%` は `0-10% family` へ、
  `12.4%` は `<0% family` まで落ちます。
  つまり `10-20%` は早い continuation entry としては使えるが、
  family classifier としては粗いです。
- speed では
  `0-10%` は `1-2日` が最良で、
  `3-5日` まで遅れると edge がいったん消えます。
  この bucket は `早く押してすぐ止まる` 形が強いと読めます。
- `10-20%` は `1-2日` でも悪くありませんが、
  `3-10日` でもむしろ改善します。
  ただし `6-10日` は `54` 件なので過信は禁物です。

## Practical Read

- 実運用に近い rule としては、
  `0-10% first pullback entry` の方が
  `family 整合性` と `path asymmetry` の両面で自然です。
- `10-20% entry` は捨てるほど悪くないものの、
  `ここで入れば upside が明らかに大きい` とまでは言いにくいです。
- したがって次段階は
  `0-10% entry` を主軸にして、
  `3-5日以降は見送る`
  あるいは
  `10-20% entry は追加条件付き`
  で refine するのが素直です。

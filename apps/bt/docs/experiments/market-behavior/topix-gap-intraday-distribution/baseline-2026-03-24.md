## 寄与の読み方

### 1. 最頻出 bucket では全群がやや弱い

- `-1% < gap < 1%` は全群で intraday 平均がマイナス。
- しかもこの bucket が全体の 86.44% を占めるため、ここでの弱さが総合期待値を強く押し下げる。
- 特に `PRIME ex TOPIX500` の中立 bucket は `-0.0286%` と最も弱い。

### 2. Moderate negative gap に対する反発は明確

- `-2% < gap <= -1%` では全群プラスだが、特に `TOPIX100 +0.2237%` と `TOPIX500 +0.2320%` が強い。
- 発生頻度も 6.64% あり、極端日よりは現実的なサンプル数がある。
- 読み: `moderate negative gap` は intraday で反発しやすく、その恩恵は大型株と TOPIX500 に出やすい。

### 3. 極端 gap は派手だが件数が少ない

- `gap <= -2%` は 3日、`gap >= 2%` は 1日しかない。
- 平均 intraday は大きくプラスだが、これだけでルール化するのは危ない。
- 読み: extreme gap は reference 程度に留め、baseline の主軸には置かない方がいい。

### 4. 頻度込みでは TOPIX100 と TOPIX500 が残る

- 中立 bucket のマイナスを抱えながらも、`-2% < gap <= -1%` の正の寄与で `TOPIX100` と `TOPIX500` は総合期待値がプラスに残る。
- `PRIME` と `PRIME ex TOPIX500` は、中立 bucket のマイナスが重く、小幅マイナスで着地する。
- 読み: gap 条件下の intraday だけを見ると、より broad な `PRIME` より、`TOPIX100` / `TOPIX500` の方が使い道が明確。

## 簡易 Rotation ルール

ルール:

- `weak => TOPIX500 long`
- `strong => PRIME ex TOPIX500 long`
- `neutral => flat`

signal の対応:

- `weak`: `gap <= -1%`
- `neutral`: `-1% < gap < 1%`
- `strong`: `gap >= 1%`

## Rotation の解釈

### 1. 実質的な alpha の中心は weak 側

- `weak => TOPIX500` の平均 return は `0.2614%` で、`strong => PRIME ex TOPIX500` の `0.1078%` を上回る。
- 読み: この固定ルールでは、上方向 gap 追随よりも、下方向 gap に対する TOPIX500 の intraday 反発の方が主役。

### 2. neutral を flat にした効果が大きい

- 86.44% を占める `-1% < gap < 1%` は全群で小幅マイナス寄りなので、ここを flat にする設計は合理的。
- 読み: この rotation は「何を買うか」より「gap がない日は何もしない」が効いている面が大きい。

### 3. 累積リターンは悪くないが、低頻度戦略である

- 取引日は 333 日で、全 2456 日のうち約 13.6% にしか入らない。
- 読み: 毎日張る戦略ではなく、gap event にだけ反応する conditional strategy として理解すべき。

## 注意点

- intraday は `open -> close` のみで、寄り付き約定スリッページや実際の執行コストは含まない。
- equal-weight の群平均であり、現実の売買可能ポートフォリオそのものではない。
- `gap <= -2%` と `gap >= 2%` は観測数が極端に少ないため、平均値の見た目ほど信頼できない。

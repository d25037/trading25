# Issues Archive

Open / active task tracking の SoT は GitHub Issues:

- https://github.com/d25037/trading25/issues

このディレクトリは local issue 時代の archive です。新規 task はここに Markdown として作成しません。

## Directories

- `done/`: GitHub Issues 移行前後の closed local issue archive
- `archive/migrated-to-github/`: 2026-04-22 に GitHub Issues へ移行した元 local open issue

長文の設計・研究計画が必要な場合は `docs/` や `apps/bt/docs/experiments/` に spec/note を置き、GitHub Issue からリンクしてください。
